  What was built

  optuna_pred_surface.py (new)

  A full 3-dimensional directional accuracy surface explorer:

  Dimensions explored:
  1. model_input_len вҖ” 5 discrete fractions of the trained input_size (35%, 50%,
   65%, 80%, 100%). Shorter windows are left-padded by neuralforecast, revealing
   how much historical context the model actually needs.
  2. test_period_days вҖ” from 30 days to the full test period (continuous QMC
  sampling), showing whether accuracy degrades or improves over different
  sub-periods.
  3. threshold вҖ” 0.0 to 0.020, answering "are stronger predictions statistically
   more accurate?"

  Two-phase workflow:
  - Cache build (pred_surface_cache.pkl): KГ—N inference calls (K=5 window
  lengths Г— N test dates). Saved once, reused.
  - Optuna QMC study (1000 trials): pure in-memory lookups вҖ” fast.

  4 output plots in <model_path>/pred_surface/:
  - pred_surface_heatmaps.png вҖ” 3 pairwise marginal heatmaps (mean accuracy over
   the 3rd axis)
  - pred_surface_lines.png вҖ” accuracy vs threshold and accuracy vs period
  length, one line per window length
  - pred_surface_hist.png вҖ” distribution of accuracy across all trials
  - pred_surface_coverage.png вҖ” accuracy + signal count vs threshold (trade-off:
   higher threshold = fewer but more accurate signals?)

  Stability badge (in UI): Stable / Moderate / Noisy based on mean вүҘ52.5% and
  std вүӨ2.5%.

  app/templates/pred_surface.html (new)

  Dark-theme page with:
  - Status panel with a "Start accuracy study" button (not auto-start, since the
   cache build takes 10вҖ“20 min)
  - Once done: sidebar with accuracy stats + signal-quality badge +
  fraction-above-chance bars + best params
  - Right panel: heatmaps + 3 further charts

  - GET /pred-surface/{folder} вҶ’ serves the page
